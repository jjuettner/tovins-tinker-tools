# actions interface
 possibility to add extra stats or display additional text in a certain situation
 depending on character stats, conditions, feats etc.



this needs to work on a generalized basis, so independent of class, feat etc.

## example:
	class feature that gives you +1d4 to damage when attacking with a spell that does fire, necrotic or poison damage

### assumption
	class feature exists in db

## requirement
	action ( attack with weapon, cast spell, receive buff, etc )
	conditions ( has feat or class feature / weapon has property, ...)
	effect (display feature text, add damage to display )
